# ReaderApp

Expressive Material 3 e-reader built with Jetpack Compose.

## Quick Start

Open the project in Android Studio and sync Gradle.

## Run (from terminal)

```powershell
cd C:\Users\rean3\AndroidStudioProjects\ReaderApp
./gradlew.bat assembleDebug
```

## Notes

- EPUB files are loaded from the `app/src/main/assets` folder.
- Reader appearance and layout settings are persisted with DataStore.
